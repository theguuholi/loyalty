# COMO CONSEGUIR 100 CLIENTES

_O passo a passo que ninguém te conta_

---

## 1. 0 → 10 CLIENTES (Aquisição inicial)

- **Foco:** Porta em porta (outbound)
- **Estratégias:**
  - Network pessoal (conhece alguém do mercado?)
  - Grupos Facebook / Reddit / Hacker News
  - Cold email via LinkedIn ✉️
  - Comprar infoproduto para acessar a comunidade ← "hack pouco conhecido"
- **OBJETIVO:** Entender objeções, refinar ICP, validar proposta de valor

---

## 2. 10 → 50 CLIENTES (Consolidação do core business)

- **Foco:** Consolidar o core business
- **Ações:**
  - Medir: quais features usam? quantas vezes? 📊
  - Ferramentas: Posthog, Amplitude
  - Entrevistar os clientes mais ativos
- **Canais novos:**
  - Continue o que está funcionando ← **NÃO ABANDONE**
  - Produção de conteúdo (médio prazo) → building in public
  - Ads para APRENDER (não para vender ainda)
- ### O MOTOR DO CRESCIMENTO
  - Ads (com competência)
  - Conteúdo (90% dos SaaS)
  - SEO

---

## 3. 50 → 100 CLIENTES (Diversificação)

- **Foco:** Diversificação
- **Estratégias:**
  - SEO + Blog (3–5 meses para resultado)
  - Ferramentas gratuitas → 🔗 "ímã de leads"
    - Ex: calculadoras, bots, extensões Chrome
  - Marketplaces: Chrome Store, App Store, Shopify
- ### ESTRATÉGIAS CORINGA 🃏
  - Parcerias com influenciadores
  - Media for equity (% por receita gerada, **NÃO** por post)
  - Eventos do nicho ← "esteja onde seu ICP está"

---

## Observações finais

- **TEMPO MÉDIO:** 8–9 meses
- Ticket alto = ciclo maior
- **Não crie expectativas irreais**
